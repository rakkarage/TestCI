# TestCI

Testing Godot deploy with github actions.

- Build HTML
  - Upload to itch.io
  - Upload to gh-pages branch
    - <https://rakkarage.github.io/TestCI/>
- Build Linux
  - Upload to itch.io
- Build Mac
  - Upload to itch.io
- Build Windows
  - Upload to itch.io
